2. Qué datos puede tratar esta web

El sitio está diseñado con un criterio de minimización de datos. En la navegación ordinaria no se solicitan datos personales obligatorios.

  • Correo electrónico en el Rincón de la Luz: se usa como puerta de cortesía para acceder a ese espacio. En la versión actual se guarda únicamente en el navegador de la persona usuaria mediante almacenamiento local y no se envía a un servidor de Peregrino APP.
  • Alias del muro de bienhechores: si se introduce un alias, se guarda únicamente en el navegador de la persona usuaria en la versión actual. No se publica en un servidor ni se comparte con terceros desde esta web.
  • Nombre o alias para una tarjeta de oración: puede aparecer en la URL de la tarjeta generada para compartirla. La persona usuaria decide si comparte ese enlace con terceros.
  • Consultas por correo electrónico: si escribes a Peregrino APP, se tratará tu dirección de correo y el contenido del mensaje para responder a tu solicitud.
Importante: evita introducir datos sensibles propios o de terceros, especialmente información de salud, situaciones familiares delicadas, datos de menores o detalles íntimos. Si quieres usar una tarjeta de oración, es preferible utilizar solo un nombre, iniciales o un alias.

3. Finalidades

  • Permitir la navegación por la web y el acceso a sus contenidos.
  • Facilitar el uso local del Rincón de la Luz: velitas, tarjetas, alias y oración mensual.
  • Responder consultas recibidas por correo electrónico.
  • Mantener la seguridad técnica y funcionamiento ordinario del sitio.

4. Base jurídica

Cuando escribes voluntariamente por correo o introduces datos en una funcionalidad local, la base puede ser tu consentimiento o la atención de una solicitud realizada por ti. Para el funcionamiento técnico básico de la web, la base puede ser el interés legítimo en mantener un sitio seguro y operativo.

5. Conservación

  • Los datos guardados en el navegador permanecen hasta que los borres, limpies el almacenamiento del navegador o uses otro dispositivo.
  • Los correos electrónicos se conservarán durante el tiempo necesario para responder y gestionar la comunicación, y después durante los plazos razonables o legales que pudieran aplicar.

6. Comunicación de datos a terceros

Esta web no vende datos personales ni los cede con fines publicitarios.

Al pulsar enlaces externos, como PayPal, WhatsApp o Google Play, pasarás a servicios de terceros que aplican sus propias políticas de privacidad. La web también se publica mediante GitHub Pages, que puede tratar datos técnicos necesarios para prestar el servicio de alojamiento.

7. Derechos

Puedes solicitar el acceso, rectificación, supresión, oposición, limitación del tratamiento y, cuando corresponda, portabilidad de tus datos escribiendo a peregrinoapp@outlook.es.

También puedes presentar una reclamación ante la Agencia Española de Protección de Datos si consideras que el tratamiento no se ajusta a la normativa aplicable.

8. Menores

Esta web no está pensada para recabar datos personales de menores. Si se utiliza una tarjeta de oración para un menor, se recomienda no incluir apellidos, datos de contacto, información de salud ni detalles identificativos.

9. Cambios en esta política

Esta política podrá actualizarse si se añaden nuevas funcionalidades, formularios, analítica, base de datos o servicios externos. La fecha de actualización aparecerá indicada al inicio de esta página.
